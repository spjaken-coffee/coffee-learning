# Bean Atlas

コーヒーチェリーの構造から、精製方法、生豆、焙煎、挽き目と抽出の科学までを学べるローカル向けのブラウザアプリです。

## コンセプト

- 図で理解する
- 自分で選んで学ぶ
- 最後に味へつなげる

特に精製方法は、分岐型のシミュレーターで選択を重ねながら結果にたどり着けるようにしています。

## 現在の MVP 範囲

- ホーム画面
- 学習マップ
- コーヒーチェリー図解
- 精製方法シミュレーター
- 焙煎の基礎チャプター
- 挽き目と抽出の基礎チャプター

## 開発

```bash
npm run dev
```

ブラウザで [http://localhost:3000](http://localhost:3000) を開くと確認できます。
